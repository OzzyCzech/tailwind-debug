# Tailwind debug plugin

Tailwind debug plugin adds a red dashed outline for the selected element.

## Installation

```bash
yarn add tailwind-debug
```

or

```bash
npm i tailwind-debug
```

## Usage

```js
// tailwind.config.js
module.exports = {
  plugins: [
    require('tailwind-debug'),
  ],
}
```
